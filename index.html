GIT-re fel

  1. mappában Git Bash Here
  2. git init //-> új repo esetén, létrejön egy rejtett .git
  3. git status //-> megmutatja a statust, vagyis van-e "feltöltetlen"
  4. git add . //-> minden file-t felküld, ezután a git status üreset listáz
  5. git add "Mappa neve" //-> ha nem mindent, akkor " " jel között megadható a file vagy mappa neve egyessével is
  6. git commit -m "Message" //-> commitoljuk a file-t üzenettel, pl első próba, talán végleges... :)

Ezután egy új Repository-t hozunk létre a Github-on pl. Citrusfelek web

  1. a Github kedves és felajánlja, hogy mit is írj továbbra a cmd-be
  2. másold ki a következőt: git remote add origin https://..../Citrusfelek web
  3. illeszd be a cmd-be és enter
  4. másold be a githubról még az előbbi alatt lévő 1 vagy 2 sort külön-külön
  5. KÉSZ!!!
